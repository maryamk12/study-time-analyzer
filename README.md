# Smart Study Time Analyzer 

Smart Study Time Analyzer is a beginner-friendly Python project that helps students track how much time they spend studying different subjects.

The user enters a subject name along with start and end times, and the program calculates the study duration in minutes and saves it to a CSV file for future reference.

---

## Features
- Accepts subject name and study time
- Validates time input (24-hour format)
- Calculates accurate study duration
- Saves study sessions to a CSV file
- Beginner-friendly and easy to extend
